Geolocation

Geolocation: Navigator

• Modern browsers have a navigator object

• Used to access/identify users' browser information

• Available in console via navigator:

navigator.language ⇒ user's language ex: en-US

navigator.platform ⇒ user's platform ex: MacIntel

navigator.language ⇒ users's browser ex: Google Inc.

Geolocation

• navigator also contains a geolocation object

• Enables users to share geolocation data with web applications

• Users must enable the feature on a per-site basis

• Note: not all browsers have this feature:


  if (navigator.geolocation) {
    /* user has geolocation */
  } else {
    /* user does not have geolocation */
  }
Geolocation

• getCurrentPosition returns users' positions

• Note: this is an asynchronous request - requires a callback

• The callback needs an arugment for the position


  navigator.geolocation.getCurrentPosition(handleResponse)

  function handleResponse(position) {
    console.log(position)
  }

  // {
  //   coords: {...}
  //   timestamp: 1521422644286
  // }

Anonymous Functions

Recap: Anonymous Functions

• Named: functions associated with a name that can be referenced later

• Anonymous: functions with no name; exist only when running; useful for callbacks


  // named function
  function hello () {
    conosle.log('hello');
  }

  // anonymous function
  function () {
    console.log('hello');
  }
Recap: Anonymous Functions

  $(document).on('click', 'li', function (event) {
    ...
  })

• The function .on() takes three arguments:

⇒ An event: click

⇒ A CSS selector: li

⇒ A callback function: function (event) {...}

• The function passed into on() does not have a name; we are passing in a raw (anonymous) function

• This is an example of treating a function as an object

Functions as First-Class Objects

Functions as First-Class Objects

• A language has first-class function objects if:

...the language supports constructing new functions during the execution of a program, storing them in data structures, passing them as arguments to other functions, and returning them as the values of other functions.

• AKA - the language treats functions just like any other data type:

⇒ Stored in a variable

⇒ Passed as a parameter to another function (callback!)

⇒ Returned from a function

⇒ Included in data structures (objects and arrays)

Functions as First-Class Objects

• Higher-order functions: functions that take other functions as an argument, or return functions

• One of JS's super powers (see: functional programming)

• JS has many higher-order functions baked in:


  setTimeout(function () {
    console.log("Hello, World!");
  }, 1000);

• setTimeout() takes two arguments:

⇒ A function to be called/invoked

⇒ The duration to wait before invoking the function

Functions as First-Class Objects

• No difference in syntax for higher-order functions:

• Open up a https://repl.it/ with:


  var blastOff = function() {
    console.log("Blasting off!");
  }

  function launchRocket(rocketName, blastOffCallback) {
    console.log("Launching " + rocketName);
    console.log("3... 2... 1...");
    blastOffCallback();
  }

  launchRocket("Viking", blastOff);

  // => Launching Viking
  // => 3... 2... 1...
  // => Blasting off!
Functions as First-Class Objects

• In the previous example, blastOffCallback is a callback

• Callbacks are functions that are called by another function

• Callbacks can take arguments:


  var blastOff = function(destination) {
    console.log("Blasting off for " + destination + "!");
  }

  function launchRocket(rocketName, blastOffCallback) {
    console.log("Launching " + rocketName);
    console.log("3... 2... 1...");
    blastOffCallback("Mars");
  }

  launchRocket("Viking", blastOff);

  // => Launching Viking
  // => 3... 2... 1...
  // => Blasting off for Mars!
Functions as First-Class Objects

Functions can also be returned from other functions:


  var blastOff = function(destination) {
    console.log("Blasting off for " + destination + "!");
  }

  function makeRocketLauncher(rocketName, blastOffCallback) {
    return function() {
      console.log("Launching " + rocketName);
      console.log("3... 2... 1...");
      blastOffCallback("Mars");
    };
  }

  var launchViking = makeRocketLauncher("Viking", blastOff);
  var launchMariner = makeRocketLauncher("Mariner", blastOff);

  launchViking();
  launchMariner();

  // => Launching Viking
  // => 3... 2... 1...
  // => Blasting off for Mars!

  // => Launching Mariner
  // => 3... 2... 1...
  // => Blasting off for Mars!
