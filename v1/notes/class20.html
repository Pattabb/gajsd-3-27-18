Course Recap: Skills Learned

JSD broke down into four units:

1) Fundamentals of JS

2) The Browser and API's

3) Data Persistence + Advanced Topics

4) Application Deployment + Extracurricular Topics

Course Recap: Skills Learned

Unit 1: Fundamentals of JavaScript:

• JS on your command line via Node.js

• Version control: git and GitHub

• Data types: string, number, boolean, etc.

• Variable assignments

• Arrays: helper methods and iterators (.forEach, .filter)

• Conditional statements, logical operators, truthy and falsy values

• Loops: for and while

• Functions: parameters/arguments; the return statement

• Data Structures: Objects + JSON

Course Recap: Skills Learned

Unit 2: The Browser and API's:

• The Document Object Model (The DOM)

• jQuery + DOM manipulation

• Browser events and event listeners: click, submit, etc.

• Handlebar.js templates

• Hypertext Transfer Protocol (HTTP)

• Asychronous JavaScript and XML (AJAX)

• Leveraging web-based API's: CitiBike, OpenWeatherMap, Reddit, Mashable, Digg

• Anonymous functions; functions as first-class objects

• OAuth (500px API)

Course Recap: Skills Learned

Unit 3: Data Persistence + Advanced Topics:

• Inheritance via prototypes

• Object construction using new

• Function scope and closures

• Context (meaning of this) and manipulation: apply, bind, and call

• CRUD using Firebase

Course Recap: Skills Learned

Unit 4: Application Deployment + Extracurricular Topics:

• Deployment with GitHub Pages + Firebase

• API Composition: Google Maps + CitiBike

• Angular.js

• Node.js + Express.js

• D3.js
