Objects: Introduction

• Objects are collections of properties, which are key:value pairs:


  {
    name: 'Obama',
    number: 44,
    titles: ['POTUS', 'Prez', 'Commander-In-Chief'],
    speak: function () {
      return 'Pay your taxes.';
    }
  }

• Useful for a simple, structured data store

• Fundamental programming paradigm that helps us structure and categorize our code

• AKA - associative array / hash / dictionary

Objects: Creation

Like arrays, two ways to create objects:

• Object Literal


	var person = {};

⇒ Useful for simple data stores

• Object Constructor


	var person = new Object();

⇒ Useful for creating many instances of a particular type

Objects: Get/Set Values

• Use either dot-notation or bracket notation to set and get propreties


  var person = {
    name: 'Obama'
  };

  // dot-notation
  person.job = 'president';

  person.job;
  // "president"

  // bracket notation (useful if a property has a special character)
  person['bad-habits'] = 'cigarettes';

  person['bad-habits'];
  // "cigarettes"

Objects: Property Iteration

• Use a for loop to iterate through an object's enumerable properties

• NOTE: unlike arrays, objects do not maintain property order


  var person = {
    name: 'Obama',
    number: 44,
    titles: ['POTUS', 'Prez', 'Commander-In-Chief'],
    speak: function () {
      return 'Pay your taxes.';
    }
  };

  for (var prop in person) {
    console.log(prop);
    console.log(person[prop]);
  }

  // name
  // "Obama"
  // number
  // 44
  // ...
Objects: Property Iteration

• Can also use Object.keys() for property iteration:


  var person = {
    name: 'Obama',
    number: 44,
  };

  Object.keys(person);
  // ["name", "number"]

  Object.keys(person).forEach(function (key) {
    var value = person[key];
    console.log(value)
  })
  // "Obama", 44

• NOTE: Object.values() is still in development

Objects: Constructors

• Constructors are factories used to create many of the same type of object

• By convention, use upper-cased variables when defining constructors

• Invoke the constructor with the new keyword


	var Person = function () {};

	var obama = new Person();
	var bush = new Person();
	var clinton = new Person();

• NOTE: JS is a prototype-based language; there are no classes!

Objects: Constructors

• Object constructors are used to set an object's properties


	var Person = function (name, age) {
	  this.name = name;
	  this.age = age;
	}

	var obama = new Person('obama', 54)

• Cleaner than creating many literals:


	var obama = {
	  name: 'obama',
	  age: 54
	}

	var bush = {
	  name: 'bush',
	  age: 70
	}
Objects: Constructors

• this is used to set a function's context

• Inside the object constructor, this refers to the current object instance

• Must use this when referring to the object using methods:


  var Person = function (name) {
    this.name = name;
    this.sayName = fucntion () {
      console.log(this.name);
    }
  }

  var obama = new Person('obama');
  obama.sayName();
  // "obama"

JavaScript Object Notation (JSON)

JSON: What is it?

• JSON: JavaScript Object Notation

• JSON !== JavaScript

• Text-based data format that is based on JS

• JSON is language-independent; Ruby, Python, etc. can all parse/generate JSON

• Successor to XML (ew)

• Easy for humans to read; easy for machines to parse

• Specified by Douglas Crockford

JSON: What's it used for?

• JSON is used as a data-exchange format between progrmams/applications

• Ex: sending data from the frontend (JavaScript), to a Python-powered backend, and vice-versa

• Has strict conventions:

⇒ Property names must be double-quoted strings: "name" vs name

⇒ All strings must be double-quoted: "name" vs 'name'

⇒ No in-line comments

⇒ Trailing commas are forbidden

JSON: JavaScript's JSON Object

Like the Math object, JavaScript has a lib for working with JSON: JSON

• JSON.stringify()

⇒ Transforms JavaScript to JSON


	var jsonObject = JSON.stringify({ name: 'obama' });
	// "{"name":"obama"}"
	typeof jsonObject
	// string

• JSON.parse()

⇒ Transforms JSON to JavaScript


	var javaScriptObject = JSON.parse(jsonObject);
	// { name: "obama" }
	typeof javaScriptObject
	// "object"
