Lab Intro

Lab Intro: Overview

Overview:

• We are building Feedr: a personalized news reader

• Our reader will pull feeds from various API's

• The user will be able to filter the feeds via a dropdown

• Clicking an article will render a modal with more info

Notes:

• This will be a single page app (SAP); all application views will be contained in the provided index.html file

• If you haven't already, read the project overview

Lab Intro: Overview

• Initially, you will pull from three news sources:

⇒ Mashable

⇒ Reddit

⇒ Digg

• Once you pull from these three, feel free to add other sources...

Lab Intro: Overview

• Please feel free to use other news API's

• Be aware that not all API's will support CORS or JSONp

• If you encounter these issues,use a proxy server to filter your API requests

• See the project overview for more info

Lab Intro: Getting Started

• Create a new directory in your github.io called Feedr

• Copy the starter code into this new directory

• Suggested First Steps:

⇒ Start by adding all the DOM functionality first

⇒ Map out the needed fields/properties from each respective feed

⇒ console.log the feeds to confirm your data before you map it

⇒ Make sure you have the JSON View chrome extension

⇒ Think about ways to best standardize all of your incoming data

⇒ Test small pieces of functionality frequently

⇒ Use tools such as Stack Overflow, Google and documentation resources to solve problems
