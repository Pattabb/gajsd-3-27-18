Data Types

Data Types

• In the machine, there is only data

• The goal of computing is to collect, manage, manipulate, and store this data

• To manipulate data, it's important to know the data type:

⇒ What the data represents

⇒ Possible values for that type

⇒ Operations that can be performed on the data

• ALWAYS KNOW YOUR DATA TYPE!

Data Types

• Numbers

42, 1337, 3.14

• Strings

Single words or sentences surrounded by quotes

'lots of kittens'

• Booleans

Either true or false values

• Arrays

Collections of related data

['manhattan', 'the bronx', 'queens', 'brooklyn', 'staten island']

Data Types

• Undefined

A variable without a value

var x = 2;

var y;

Variable y is undefined

• Null

Used to indicate no value

var x = null

• Note: undefined ≠ null

• More info on null vs. undefined

Data Types

Variables

Variables

• Variables keep track of data in a program

• AKA - managing a program's state

• Use the var keyword to declare a variable assignment

var num = 1337;

• Stores this piece of data in memory under num

• Use the variable's name to "call", or retrieve, its value from memory

Variables

• Take care when naming variables

• Names should reflect the data they describe

• Other naming conventions

⇒ Names should start with a lowercase letter

⇒ Cannot use spaces; use hyphens or underscores instead

⇒ Better yet, use camelCase:


	var currentTemp = 82;

	var targetTemp = 72;
Variables - Reassignment

1. Create a variable, x, and assign it to 2


	var x = 2;

2. Perform some arithmetic on x


	x * 2
	// 4

3. What's x now equal to?


	x
	// 2

Variables - Reassignment

• You must reassign variables to change them


	var x = 2;
	// 2

	x * 2
	// 4

	x
	// 2

	// REASSIGN
	x = x * 2
	// 4

	x
	// 4

Arrays

Arrays

• Arrays are a list-like data structure of comma-separated values

• Useful for storing a sequence of related data

• Declare arrays using square brackets: [ ]


	var friends = ['Moe', 'Larry', 'Curly']

• Element - an item inside the array

• Index - the numerical location of each element in the array (starts at position 0 - aka "0-indexed")

Arrays

• JS provides multiple methods for array creation:

⇒ The new keyword


	var drinks = new Array('beer', 'coffee', 'water')

⇒ Using the array literal


	var drinks = ['beer', 'coffee', 'water']

• While similar, there are crucial differences

• We will be using the array literal in JSD

Arrays

• Use [index] to retrieve values from an array

• Remember that arrays are 0-indexed


	var drinks = ['beer', 'coffee', 'water']

	drinks[0]
	// 'beer'

	drinks [0+1]
	// 'coffee'

	var x = drinks.length - 1
	drinks[x]
	// 'water'

	drinks[4]
	// undefined
Arrays

• To add a value to END of an array, use the .push() method


	var drinks = ['beer', 'coffee', 'water']

	drinks.push('wine')
	// 4

	drinks
	// [ 'beer', 'coffee', 'drinks', 'wine' ]

• To add a value to the BEGINNING, use .unshift()


	drinks.unshift('tea')
	// 5

	drinks
	// [ 'tea', 'beer', 'coffee', 'drinks', 'wine' ]

Array Helper Methods

Array Helper Methods

• JavaScript provides tools to make working with arrays easier:

.push()

.pop()

.shift()

.unshift()

.reverse()

• More methods here

Array Helper Methods

• First, let's create an array with some values:


  var message = []

  message.push(8)
  // 1

  message.push('r', 'e', 'b', 'm', 'u')
  // 6

  message.push('n', 's', 'i', 'A', 'G', 'K')
  // 12

  message
  // [ 8, 'r', 'e', 'b', 'm', 'u', 'n', 's', 'i', 'A', 'G', 'K' ]
Array Helper Methods

Use .pop() to remove, and return, the last value:


  message.pop()
  // 'K'

Use .shift() to remove, and return, the first value:


  message.shift()
  // 8

Use .unshift() adds values to the beginning:


  message.unshift(1)
  // 11
Array Helper Methods

Use .reverse() to reverse an array


  message.reverse()
  // [ 'G', 'A', 'i', 's', 'n', 'u', 'm', 'b', 'e', 'r', 1 ]

Use .join() to join all elements into a string


	message.join()
	// 'G,A,i,s,n,u,m,b,e,r,1'

	message.join(' ')
	// 'G A i s n u m b e r 1'

Note: .join() takes an agrument, the separator, which becomes a string that separates the values

Array Iteration

Array Iteration

• Consider an array of Tweets; how does a developer render an unknown length of Tweets for a user to see?

Array Iteration

• JS arrays have several advanced iteration methods:

.forEach()

.every()

.some()

.filter()

.map()

• Most require a function to be supplied as an argument; this function will be applied to each item in the array, individually

Array Iteration

for vs. Array.forEach


  var boroughs = ['manhattan', 'the bronx', 'queens', 'brooklyn', 'staten island']

  for (var i = 0; i < boroughs.length; i++) {
	var borough = boroughs[i]
	console.log(borough)
  }

  // 'manhattan'
  // 'the bronx'
  // 'queens'
  // ...



  var boroughs = ['manhattan', 'the bronx', 'queens', 'brooklyn', 'staten island']

  boroughs.forEach(function (borough) {
	console.log(borough)
  })

  // 'manhattan'
  // 'the bronx'
  // 'queens'
  // ...
Array Iteration

• First, let's create two arrays with values:


  var evens = []

  evens.push(2,4,6,8,10)
  // 5

  var odds = []

  odds.push(1,3,5,7,9)
  // 5

• Note: the % operator (modulus)

Returns the remainder; useful for determining odds/evens

5 % 2 = 1

4 % 2 = 0

Array Iteration

Array.every()

⇒ Tests whether all elements in the array pass the test implemented by the provided function


  evens.every(function (num) {
    return num % 2 === 0
  })
  // true

  evens.every(function (num) {
    return num % 4 === 0
  })
  // false
Array Iteration

Array.some()

⇒ tests whether some elements in the array pass the test implemented by the provided function


  evens.some(function (num) {
    return num % 4 === 0
  })
  // true
Array Iteration

Array.filter()

⇒ creates a new array with all elements that pass the test implemented by the provided function; does not mutate


  evens.filter(function (num) {
    return num > 5
  })
  // [6, 8, 10]

  evens
  // [ 2, 4, 6, 8, 10 ]

  var bigNums = evens.filter(function (num) {
    return num > 5
  })
  // undefined

  bigNums
  // [6, 8, 10]

  var smallNums = odds.filter(function (num) {
    return num < 5
  })
  // undefined

  smallNums
  // [1, 3]
Array Iteration

Array.map()

⇒ creates a new array with the results of calling a provided function on every element in this array; does not mutate


  var timesFive = evens.map(function (num) {
    return num * 5
  })
  // undefined

  timesFive
  // [10, 20, 30, 40, 50]

  var timesTen = odds.map(function (num) {
    return num * 10
  })
  // undefined

  timesTen
  // [10, 30, 50, 70, 90]
